Predeterminado

Arriba 
get /api/calcularImpacto
Obtener todos los resultados de cálculo de impacto (apiCalcularImpactoGet)

Tipo de retorno

array[EntidadValor]

Datos de ejemplo

Content-Type: application/json
[ {
  "entidad_id" : 0,
  "gradoImpacto" : 6.027456183070403
}, {
  "entidad_id" : 0,
  "gradoImpacto" : 6.027456183070403
} ]

Produce

Esta llamada a la API produce los siguientes tipos de medios de acuerdo con la solicitud de encabezado Accept; el tipo de medio se transmitirá mediante el encabezado de respuesta Content-Type.
  • application/json

Respuestas

200

Lista de resultados de cálculo de impacto

404

No se encontraron resultados Error
Arriba 
post /api/calcularImpacto
Calcular el impacto de una entidad (apiCalcularImpactoPost)

Consumir

Esta llamada a la API consume los siguientes tipos de medios a través del encabezado de solicitud Content-Type:
  • application/json

Cuerpo de solicitud

body ListadoValores (requerido)
Parámetro de cuerpo —

Tipo de retorno

Integer

Datos de ejemplo

Content-Type: application/json
0

Produce

Esta llamada a la API produce los siguientes tipos de medios de acuerdo con la solicitud de encabezado Accept; el tipo de medio se transmitirá mediante el encabezado de respuesta Content-Type.
  • application/json

Respuestas

200

Cálculo exitoso Integer

404

No se pudo calcular el impacto Error

500

Error inesperado Error
